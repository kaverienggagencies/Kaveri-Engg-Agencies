# KAVERI ENGG AGENCIES Website - Separate Images Version

This version has a clean folder structure:

- index.html
- assets/images/

Use this version for GitHub Pages and mobile reliability.

## How to upload to GitHub
1. Extract this ZIP.
2. Open your GitHub repository.
3. Upload `index.html`.
4. Upload the full `assets` folder also.
5. Commit changes.
6. Wait 1-5 minutes and refresh your live website.

Do not upload only index.html. This version requires the `assets/images` folder.
